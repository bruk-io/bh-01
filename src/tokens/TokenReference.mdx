# Token Reference

This is the **contract** that all themes must implement. Components only reference these semantic token names — never primitives directly.

To create a new theme, provide values for every token listed below.

---

## Creating a Custom Theme

```css
/* my-theme.css */
@import '@bruk-io/bh-01/tokens';  /* primitives */

:root,
[data-theme='my-brand'] {
  /* Surfaces */
  --bh-color-surface: #fafafa;
  --bh-color-surface-raised: #ffffff;
  --bh-color-surface-overlay: #ffffff;

  /* Text */
  --bh-color-text: #1a1a1a;
  --bh-color-text-muted: #737373;
  --bh-color-text-inverse: #ffffff;

  /* Primary — use your brand color */
  --bh-color-primary: #e11d48;
  --bh-color-primary-hover: #be123c;
  --bh-color-primary-active: #9f1239;
  --bh-color-primary-text: #ffffff;

  /* ... fill in all remaining tokens */
}
```

Import your theme instead of the default, and all components adapt automatically.
