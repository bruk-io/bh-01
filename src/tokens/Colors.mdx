# Color Primitives

Named color palette. These are **never used directly in components** â€” they feed into semantic tokens via the theme.

## Accents

<div style={{ display: 'grid', gridTemplateColumns: 'repeat(auto-fill, minmax(140px, 1fr))', gap: '0.75rem' }}>
  {[
    ['mandarin', '#FF6B35'],
    ['sky', '#5DAFD6'],
    ['pomegranate', '#F83E1F'],
    ['amaranth', '#E92C63'],
  ].map(([name, hex]) => (
    <div key={name} style={{ display: 'flex', flexDirection: 'column', gap: '0.25rem' }}>
      <div style={{ width: '100%', height: '3rem', borderRadius: '0.375rem', background: `var(--bh-color-${name})`, border: '1px solid var(--bh-color-border)' }} />
      <code style={{ fontSize: '0.7rem' }}>--bh-color-{name}</code>
      <span style={{ fontSize: '0.6rem', color: 'var(--bh-color-text-muted)' }}>{hex}</span>
    </div>
  ))}
</div>
